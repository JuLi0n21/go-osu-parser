# OsuParser

Small go based Library to Read osu! dbs and .osu files

Currently Supported files:
-   osu!.db
- scores.db
- collection.db

- .osu files

Planned features 
- Reading ReplayFiles

- Changing and Saving afformentioned files

For usage information look at the `pkg/main.go` examples
